# 🏡 FamilyTasks

**FamilyTasks** — это веб-приложение для организации домашних задач внутри семьи с элементами геймификации: баллы, стрики и рейтинг участников.

---

## 🛠 Технологии

* **Backend:** Flask, SQLAlchemy
* **Auth:** Flask-Login
* **Database:** PostgreSQL
* **Frontend:** HTML, Jinja2, CSS
* **Infrastructure:** Docker, Docker Compose

---



## 🚀 Быстрый старт (Docker)

### 1. Клонировать репозиторий

```bash
git clone https://github.com/Vorobey4K/familytasks-flask-docker.git
cd familytasks-flask-docker
```

### 2. Настроить переменные окружения

```bash
cp .env.example .env
```
# Заполните .env своими значениями

### 3. Запустить проект

```bash
docker compose up --build
```

### 3. Открыть в браузере

```
http://localhost:8000
```

## ✨ Основные возможности

* 🔑 Регистрация и авторизация пользователей
* 🧺 Добавление выполненных задач и получение баллов

* 🎯 Система баллов и ежедневных стриков

## 📊 Страницы приложения

| Страница                         | Описание                             |
| -------------------------------- | ------------------------------------ |
| `/dashboard`                     | Добавление задач и начисление баллов |
| `/profile`                       | Личный кабинет и статистика          |
| `/family`                        | Рейтинг семьи                        |
| `/login`, `/register`            | Аутентификация                       |
| `/edit_profile`                  | Редактирование профиля               |
| `/create_family`, `/join_family` | Работа с семьями                     |
